# Sistema-de-Livraria
Este programa implementa um sistema simples de gerenciamento de obras literárias utilizando arquivos binários em C dentro de um código C++. 
Ele permite:  
✔ Cadastrar livros
✔ Listar o primeiro livro cadastrado
✔ Procurar livros pelo código 
✔ Armazenar dados em arquivo binário (livros.bin).
